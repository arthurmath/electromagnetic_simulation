# Magnetic Field Visualisation App

App simulating the magnetic field of coils and magnets. React frontend contains an interactive visualization, python codes generates images.


## Frontend React

Required: NodeJS (mac: "brew install node")

Setup:

1. Navigate to the frontend directory:
   cd frontend

2. Install dependencies:
   npm install

3. Start the development server:
   npm run dev

4. Open your browser to:
   http://localhost:3000

5. Quit:
   Press q + enter



## Python : 

Required: Python 3.10+

Setup:

1. Create virtual environnement:
   python3 -m venv .venv  
   
2. Activate it:
   source .venv/bin/activate  

3. Install dependencies:
   pip install -r requirements.txt  

4. Generate images:
   python3 simulation.py
